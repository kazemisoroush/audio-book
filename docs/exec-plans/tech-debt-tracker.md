# Tech Debt Tracker

Items here are known suboptimal patterns that should be addressed. Each item
has a severity (low / medium / high) and a description of the ideal state.

On a regular cadence, a cleanup agent scans for deviations and opens
targeted fix PRs.

## How to add an item

```
### TDT-NNN: Short description
- **Severity**: low | medium | high
- **Location**: file or module path
- **Current state**: what it does now
- **Ideal state**: what it should do
- **Why deferred**: the reason it wasn't fixed when discovered
```

## How to close an item

Move it to the Closed section below with the PR that fixed it.
