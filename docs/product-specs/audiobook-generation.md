# Product Spec: Audiobook Generation

## What this is

A CLI tool that takes a plain-text book file (Project Gutenberg `.txt` format)
and produces a multi-voice audio file where different characters speak in
distinct voices generated by ElevenLabs TTS.

## User journey

1. User has a `.txt` file (e.g., `books/pg1342.txt` — Pride and Prejudice)
2. User runs: `python -m audiobook.cli books/pg1342.txt --discover-characters`
   - Output: a list of detected characters and their assigned ElevenLabs voices
   - User can accept or edit the mapping in a generated `books/pg1342.voices.json`
3. User runs: `python -m audiobook.cli books/pg1342.txt`
   - The tool reads the voice mapping (or creates a default one)
   - Splits the book into segments: narration vs. character dialogue
   - Calls ElevenLabs TTS for each segment with the appropriate voice
   - Assembles all audio segments into a single output file
   - Output: `books/pg1342.mp3` (or configured output path)

## Acceptance criteria

### Character discovery (`--discover-characters`)
- Identifies at least the top 10 characters by dialogue frequency
- Assigns each a unique ElevenLabs voice ID
- Writes a JSON file the user can inspect and edit before generation
- Runs without making any ElevenLabs API calls (discovery is local)
- Completes in under 30 seconds for a standard Gutenberg novel

### Audiobook generation
- Handles narration (no speaker tag) vs. dialogue (attributed to a character)
- Uses the narrator voice for all non-dialogue text
- Calls ElevenLabs API with retry on rate-limit (429) and transient errors
- Caches generated audio segments — re-runs skip segments already generated
- Assembles segments in order using ffmpeg
- Progress is shown as a percentage bar to stdout

### Error handling
- Invalid API key: exits immediately with a clear message
- Network error after retries: exits with the segment index that failed (allows resume)
- Missing voice ID in mapping: warns and falls back to narrator voice
- Corrupted cache: detected and regenerated automatically

## Out of scope (v1)
- Web UI or API server
- Multi-book batch processing
- Real-time streaming audio
- Voice cloning or custom voice creation
- Languages other than English
